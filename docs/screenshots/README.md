Drop README media here. Filenames the README expects:

- `home.png` — the landing / create-join screen
- `lobby.png` — the lobby (show the voice section)
- `game.png` — in-game drawing & guessing
- `leaderboard.png` — the leaderboard
- `banner.png` — optional hero banner

For the demo video, you don't need a file here — drag-and-drop an `.mp4`
directly into the README on GitHub (it uploads to `user-attachments`), or
link a YouTube clip. Then replace the `REPLACE_WITH_YOUR_DEMO_VIDEO`
placeholder near the top of `README.md`.
